# 核心改动
- 将 `JsonHelper.JsonCompress` 改为先解析再输出单行 JSON，避免非法内容被字符级“压缩”后误判成功
- 新增 `EscapeJsonText` 与 `UnescapeJsonText`，统一用 JSON 语义处理转义与去转义，避免手工 `Replace` 破坏内容
- 调整 `JsonPageViewModel.ReplaceEscape`，仅在完整处理成功后回写结果，失败时保留用户原始输入
- 删除 `JsonPageView` 中与 `TextEditorBinding` 重复的 `TextChanged` 手动同步逻辑，避免双重回写
- 补充 `.learnings/ERRORS.md`，记录 `JsonException` 命名冲突的编译问题
