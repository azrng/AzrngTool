# JSON页右侧宿主滚动条隐藏

## 本次目标
- 去掉 JSON 页面右侧额外出现的纵向滚动条，减少页面外层滚动干扰。

## 核心改动
- 将主窗口内容宿主 `ScrollViewer` 的纵向滚动条可见性改为 `Hidden`，保留滚动能力但不再显示右侧额外滚动条。
- 将 JSON 页面外层 `ScrollViewer` 的纵向滚动条可见性同步改为 `Hidden`，避免页面再叠加一层外部滚动条。
- 保留 JSON 编辑器内部自己的纵向滚动条，继续支持长文本编辑。

## 修改文件
- `AzrngTools/Views/MainWindow.axaml`
- `AzrngTools/Views/Format/JsonPageView.axaml`
- `TASK.md`

- 结果：通过，0 warning / 0 error

- 结果：命令执行完成

## 风险或遗留项
- 这次隐藏的是宿主和页面外层滚动条，若个别旧页面仍依赖宿主滚动条作为主要视觉提示，需在实际界面中再补一轮人工检查。
