# 2026-04-01 更新检查 SSL 兼容修复

## 本次目标
- 修复关于页检查更新时与 GitHub 建立 SSL/TLS 连接失败的问题

## 核心改动
- 更新 `AzrngTools/App.axaml.cs`
- 为更新服务注册命名 `HttpClient`
- 将默认请求版本固定为 HTTP/1.1，并启用常见压缩格式解压
- 重写 `AzrngTools/Services/AppUpdateService.cs`
- 为更新检查和更新包下载增加兼容模式重试
- 首次请求失败时，自动回退到更保守的 `HttpClientHandler` 配置重试
- 在兼容模式中显式启用 TLS 1.2 / TLS 1.3，并关闭证书吊销检查以提升部分环境下的可连通性
- 增加日志记录与更明确的用户提示，避免仅出现底层 SSL 英文异常
