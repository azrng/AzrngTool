# 2026-03-26 样式系统与 .NET 10 升级

## 本次目标
- 统一项目样式资源，消除页面中的高频硬编码样式
- 将项目目标框架升级到 .NET 10
- 让 AI 规则文件与当前工具箱项目保持一致

## 核心改动
- 新增 `AzrngTools/Styles/DesignTokens.axaml`，集中维护间距、圆角、字号、常用画刷和布局尺寸资源
- 新增 `AzrngTools/Styles/Global.axaml`，统一放置全局样式规则
- 将页面中的高频 `Margin`、`Padding`、`CornerRadius`、`FontSize`、颜色和 `Spacing` 改为资源引用
- 调整 `App.axaml` 的资源接入方式，拆分 token 资源与全局样式
- 将项目目标框架升级到 `net10.0-windows`
- 重写 `AGENTS.md`、`CLAUDE.md`、`GEMINI.md` 与 `design-system.yaml`，使其匹配当前工具箱项目
- 顺手移除 `OverviewPageView` 直接 new ViewModel 的做法，并让主窗口 ViewModel 通过 DI 获取首页实例
