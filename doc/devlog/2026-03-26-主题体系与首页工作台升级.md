# 2026-03-26 主题体系与首页工作台升级

## 本次目标
- 为项目接入更现代的主题底座，修复亮色模式下的可读性问题，并把首页改成更有层次的概览工作台。

## 核心改动
- 引入 `FluentAvaloniaUI`，并将应用主题底座从原生 `FluentTheme` 切换为 `FluentAvaloniaTheme`。
- 同步升级 Avalonia 相关包版本，并保留 `Semi.Avalonia` 作为兼容主题资源层。
- 在 `DesignTokens.axaml` 中新增亮色/暗色语义化颜色 token、应用字体 token 和首页 Hero 尺寸 token。
- 重写 `Global.axaml`，统一窗口、壳层、卡片、导航选中态和概览页样式。
- 调整主窗口左侧导航分段，让“常用入口”和“全部工具”更容易辨识。
- 重做首页 `OverviewPageView`，增加 Hero 区、统计卡、说明卡和升级说明信息条。
- 顺手修复 `ClipboardHelper` 的过时 API 调用，让构建恢复为 0 warning。
