# 2026-03-28 侧边栏常用入口动态化

## 本次目标
- 移除左侧侧边栏里的筛选标签区域
- 将常用入口改为按使用频率动态计算，默认最多展示 3 个入口
- 将使用记录持久化到用户数据目录，便于后续持续计算常用工具

## 核心改动
- 删除 `MainWindow` 左侧的筛选标签 UI 和对应筛选逻辑，侧边栏改为 `搜索 -> 统计 -> 常用入口 -> 分组列表`
- 将常用入口改为“首页 + 最多 2 个动态常用工具”，总数默认不超过 3 个
- 新增 `IToolUsageStatsService` / `ToolUsageStatsService`，把使用记录保存到用户目录下的 `tool-usage-stats.json`
- 在 `MainWindowViewModel` 中记录工具打开次数，并在每次打开工具后重新刷新常用入口排序
- 建议算法采用 `频率平滑 + 最近使用衰减`：
  `score = log2(UseCount + 1) * 0.75 + exp(-daysSinceLastUse / 7) * 0.25`
- 当历史记录不足时，回退到预设推荐工具：`GUID生成器 / JSON格式化 / URL编码解码`
