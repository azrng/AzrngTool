# JSON页宿主滚动条修复

## 本次目标
- 修复 JSON 页面右侧工作区仍被宿主滚动容器撑出滚动条的问题。

## 核心改动
- 去掉 JSON 编辑器固定的较大最小高度，避免页面内容天然超出宿主视口。
- 为 JSON 页面增加宿主视口高度监听，进入页面后根据可见区域动态限制工作区卡片和编辑器高度。
- 保留编辑器内部滚动能力，让长文本继续在编辑器内滚动，而不是把外层页面继续撑高。

## 修改文件
- `AzrngTools/Views/Format/JsonPageView.axaml`
- `AzrngTools/Views/Format/JsonPageView.axaml.cs`
- `TASK.md`

## 风险或遗留项
- 当前高度计算采用宿主视口减固定边距的方式，若后续 JSON 页顶部结构再次变化，可能需要微调减去的留白值。
