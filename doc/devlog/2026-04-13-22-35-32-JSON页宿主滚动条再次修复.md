# JSON页宿主滚动条再次修复

## 本次目标
- 继续修复 JSON 格式化页面右侧仍然存在的宿主级滚动条，保留编辑器内部滚动能力。

## 核心改动
- 移除 `JsonPageView` 自身的外层 `ScrollViewer`，避免与主窗口内容区形成嵌套滚动容器。
- 将页面根布局改为固定可视区高度分配，按宿主视口高度和页面头部实际高度动态计算工作区高度。
- 不再单独硬编码编辑器高度，改为由工作区卡片高度驱动内部编辑器自然拉伸，减少外层高度叠加导致的溢出。
