# JSON页输入同步修复

## 本次目标
- 移除 JSON 编辑区上方冗余提示条。
- 修复 AvaloniaEdit 当前输入内容未及时同步到格式化命令，导致点击按钮后误提示“请输入要处理的内容”的问题。

## 核心改动
- 删除编辑区标题下方的说明文本和浅色提示条，减少无效占位。
- 在 `JsonPageView` 中监听 `TextEditor.TextChanged`，实时把编辑器内容同步回 `JsonPageViewModel.Original`。
- 保留原有格式化、压缩、转义和去除转义命令逻辑，只修复输入同步链路。
