# 2026-04-08 版本号切换为三段式

## 本次目标
- 将当前版本体系从历史四段式迁移为三段式
- 保持 `VERSION` 作为唯一版本来源
- 保证历史四段版本仍可参与更新比较

## 核心改动
- 将根目录 `VERSION` 从 `2026.04.08.1221` 调整为三段式 `2026.4.9`
- 在 `AzrngTools.csproj` 中显式设置 `AssemblyVersion`、`FileVersion` 和 `InformationalVersion`
- 调整 `AppInfoService`，让应用展示版本优先输出三段主版本号，而不是默认四段程序集版本
- 调整 `AppUpdateService` 的版本提取逻辑，使其同时兼容三段与历史四段版本字符串
- 更新 `README.md` 中的版本号规则说明，明确当前三段式方案与历史兼容策略
